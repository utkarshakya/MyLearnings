# Backend Learning Repository

A collection of backend development concepts and implementations.

## Prerequisites

- Node.js (v14+)
- MongoDB
- Postman (for API testing)
- Basic JavaScript knowledge

## Project Structure

```plaintext
Backend/
├── 01_NodeJs/        # Basic Node.js concepts
├── 02_NodeJS/        # Node.js servers
├── 03_ExpressJs/     # Express framework
├── 04_MongoDB/       # Database operations
└── 05_More/          # JWT, Redis, Rate limiting
```

## How to Use

1. **Setup**:
   - Clone the repository
   - Install Node.js if not already installed
   - Install MongoDB locally or use MongoDB Atlas

2. **Running Projects**:
   - Navigate to specific project folder
   - Run `npm install` to install dependencies
   - Follow instructions in individual project folders

3. **Order of Learning**:
   - Start with NodeJS basics (01_NodeJs)
   - Move to server creation (02_NodeJS)
   - Learn Express framework (03_ExpressJs)
   - Study database operations (04_MongoDB)
   - Explore advanced topics (05_More)
